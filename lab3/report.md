# Отчет по Задаче 3: ИИ-агент "Фитнес-тренер"

## 1. Общая информация
**Роль агента:** Фитнес-тренер и спортивный нутрициолог.  
**Задача:** Составление программ тренировок, подбор упражнений, расчет базового обмена веществ (BMR) и рекомендации по восстановлению.  
**Технический стек:** Python 3.13, LangChain v1.x, LangGraph, FastAPI, HTML/JS (Client-side).  
**LLM:** `qwen/qwen3-4b-2507` (локальная модель через LM Studio).

## 2. Архитектура решения
Решение построено как единый сервис, объединяющий логику агента и веб-интерфейс.

1.  **Бэкенд (FastAPI + LangChain):**
    *   **Файл `agent.py`:** Содержит логику инициализации модели, определение инструментов (`@tool`) и конфигурацию агента. Для взаимодействия с LM Studio используется провайдер `ChatOpenAI` (так как LM Studio эмулирует OpenAI API).
    *   **Агент:** Используется `create_react_agent` из `langgraph`. Это современный подход к созданию агентов с циклом "Мысль -> Действие -> Наблюдение".
2.  **Фронтенд (HTML/JS):**
    *   **Файл `inference.html`:** Одностраничное приложение (SPA). Отдается напрямую через FastAPI.
    *   **Особенности:** Поддержка Markdown-разметки (через `marked.js`), защита от XSS (`DOMPurify`), анимация набора текста, быстрые запросы.
3.  **Сервер (Uvicorn):**
    *   Запускает API для инференса (`POST /generate`) и раздает статику.

## 3. Реализованные инструменты (Skills)
Агент оснащен двумя инструментами, которые он вызывает автоматически при необходимости:

| Инструмент | Описание | Аргументы |
| :--- | :--- | :--- |
| `calculate_bmr` | Рассчитывает базовый обмен веществ и суточную калорийность по формуле Миффлина-Сан Жеора. | `weight`, `height`, `age`, `gender` |
| `get_exercises_for_muscle` | Возвращает готовую программу тренировок (упражнения, подходы, повторения) для конкретной группы мышц. | `muscle_group` (строка) |

## 4. Системный промпт и ограничения
В системный промпт жестко зашиты правила поведения:
*   **Специализация:** Только фитнес, тренировки, питание спортсменов.
*   **Обработка неизвестных команд:** Если запрос не относится к теме (например, "напиши стих" или "рецепт борща"), агент обязан ответить строго: **"Команда неизвестна"**.
*   **Стиль ответа:** Структурированный, на русском языке, с использованием Markdown.
